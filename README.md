# Based bot

Бот с нескучными демотиваторами. В перспективе возможны другие фронтенды помимо вконтакте. Работает (на момент написания ридми) вот здесь: https://vk.com/kallinux

#### Функции:
- Демотиваторы из пересланных сообщений/картинок/полностью автоматические
- Споры для редактора http://objection.lol/maker из пересланных сообщений
- Генерация предложений от известного политика
- Поддержка запуска интерпретатором [PyPy](https://pypy.org)
- Другие мелочи

## Как использовать?
#### Подготовка
Необходимо установить `docker-compose`.

Боту требуются права на управление сообществом, сообщения сообщества, фотографии и документы.

#### Запуск
```
VK_BOT_TOKEN=<токен группы> docker-compose -f docker-compose.yml up --build
```
#### Firebase Realtime Database
Бот поддерживает отправку статистики 1 беседы VK в Firebase RTDB. Отправляться могут следующие данные:
- количество участников
- список участников онлайн (активных в беседе менее чем 5 минут назад)
- топ участников по количеству сообщений с момента начала сбора статистики
- список админов

Включить сбор статистики можно объявлением следующих переменных окружения:
```
STATS=1
STATS_CHAT=<id беседы>
FIREBASE_DB_URL=<URL базы данных Firebase>
FIREBASE_KEY_PATH=<расположение .json файла с ключом доступа Firebase Admin SDK>
```
Пример использования на вебсайте: https://librespeak.ru
